# Sales Data Validation and Performance Dashboard (Excel)
Sales data analysis and dashboard project using Excel, featuring data validation, cleaning, and interactive reporting. Provides insights on top-performing agents, product performance, and sales trends to support data-driven business decisions.

### Overview
This project develops a structured end-to-end data workflow for a sales environment, integrating data entry, cleaning, analysis, and visualization. The objective is to improve data accuracy, streamline reporting, and support effective business decision-making.

---

### Data Validation and Entry Design
A data validation system was designed to ensure consistent and accurate data entry by the sales team. Key features include:
- Standardized formats for IDs and ticket numbers using Excel functions (e.g., LEFT)
- Dropdown lists for controlled inputs such as agent names
- Input guidance and error alerts to reduce entry mistakes  

---

### Data Cleaning and Preparation
The dataset was prepared by:
- Removing duplicates  
- Identifying and correcting inconsistencies  
- Standardizing date formats  
- Resolving ticket ID errors  

*Note: The dataset is not original. The validation structure and workflow were independently designed and implemented.*

---

### Dashboard Development
An interactive Excel dashboard was created to track and communicate key business metrics:
- Goals set vs. targets achieved  
- Top-performing agents  
- Best-selling products  

Slicers for **Agent** and **Location** were integrated across all pivot tables, enabling dynamic filtering and deeper analysis (e.g., tracking product sales by location and agent performance).

---

### Key Insights
- Roch Cousineau is the top-performing agent by total sales  
- Adrien Martin has the highest performance rating  
- Paneer Tikka Pizzabun is the best-selling product, followed by Crispy Chole Pizza  
- Sales performance varies by location, with certain products performing better in specific regions  

---

### Business Implications
- High-demand products such as Paneer Tikka Pizzabun present strong opportunities for expansion and product focus  
- Assigning top-performing agents to new or strategic locations can improve revenue generation  
- Implementing structured data validation improves data quality and supports more reliable analysis and reporting  
- Linking data entry, analysis, and visualization creates a more efficient and scalable business workflow
